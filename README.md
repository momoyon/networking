# Networking

An graphical application to learn networking.

## Usage
```bash
$ make networking
$ ./networking
```

## Dependencies
- [commonlib.h](https://github.com/momoyon/commonlib)
- [raylib-5.0](https://github.com/raysan5/raylib/releases/tag/5.0)
- [stb\_ds.h](https://github.com/nothings/stb)
